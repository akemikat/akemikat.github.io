# akemikat.github.io

Personal portfolio site for **Kat Korson**, Founder & Technical Director at [Red Eagle Tech](https://redeagle.tech).

## About

A single-page dark-themed portfolio covering:

- **About** - background and what I do
- **Projects** - Red Eagle Tech, MeldEagle, CDR API
- **Tech stack** - languages, cloud, data, AI, and testing tools
- **Writing** - featured articles from the [Red Eagle Tech blog](https://redeagle.tech/blog)

## Built with

- HTML & CSS (no frameworks, no build step)
- [Noto Sans](https://fonts.google.com/noto/specimen/Noto+Sans) (headings) and [Lato](https://fonts.google.com/specimen/Lato) (body) via Google Fonts
- Hosted on [GitHub Pages](https://pages.github.com/)
